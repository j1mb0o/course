## Χρήση notebook Google Colab[[using-a-google-colab-notebook]]

Η χρήση ενός notebook Colab είναι η πιο απλή δυνατή ρύθμιση· ανοίξτε ένα notebook στο πρόγραμμα περιήγησής σας και ξεκινήστε αμέσως να γράφετε κώδικα!

Αν δεν είστε εξοικειωμένοι με το Colab, σας συνιστούμε να ξεκινήσετε ακολουθώντας την [εισαγωγή](https://colab.research.google.com/notebooks/intro.ipynb). Το Colab σάς επιτρέπει να χρησιμοποιείτε κάποιο επιταχυντικό υλικό, όπως GPU ή TPU, και είναι δωρεάν για μικρότερα φορτία εργασίας.

Αφού εξοικειωθείτε με το Colab, δημιουργήστε ένα νέο notebook και ξεκινήστε με τη ρύθμιση:

<div class="flex justify-center">
<img src="https://huggingface.co/datasets/huggingface-course/documentation-images/resolve/main/en/chapter0/new_colab.png" alt="An empty colab notebook" width="80%"/>
</div>

Το επόμενο βήμα είναι να εγκαταστήσετε τις βιβλιοθήκες που θα χρησιμοποιήσουμε σε αυτό το μάθημα. Θα χρησιμοποιήσουμε το `pip` για την εγκατάσταση, που είναι ο διαχειριστής πακέτων για Python. Στα notebooks, μπορείτε να εκτελείτε εντολές συστήματος προηγώντας τους τον χαρακτήρα `!`, οπότε μπορείτε να εγκαταστήσετε τη βιβλιοθήκη 🤗 Transformers ως εξής:

```
!pip install transformers
```

Μπορείτε να βεβαιωθείτε ότι το πακέτο εγκαταστάθηκε σωστά εισάγοντάς το στο Python runtime σας:

```
import transformers
```

</div>

Αυτό εγκαθιστά μια πολύ ελαφριά έκδοση του 🤗 Transformers. Συγκεκριμένα, δεν εγκαθίστανται συγκεκριμένα πλαίσια μηχανικής μάθησης (όπως PyTorch ή TensorFlow). Επειδή θα χρησιμοποιήσουμε πολλές διαφορετικές λειτουργίες της βιβλιοθήκης, συνιστούμε να εγκαταστήσετε την έκδοση ανάπτυξης, η οποία συνοδεύεται από όλες τις απαραίτητες εξαρτήσεις για σχεδόν κάθε φανταστή περίπτωση χρήσης:

```
!pip install transformers[sentencepiece]
```

Αυτό θα πάρει λίγο χρόνο, αλλά έπειτα θα είστε έτοιμοι για το υπόλοιπο μάθημα!

## Χρήση εικονικού περιβάλλοντος Python[[using-a-python-virtual-environment]]

Αν προτιμάτε να χρησιμοποιήσετε εικονικό περιβάλλον Python, το πρώτο βήμα είναι να εγκαταστήσετε Python στο σύστημά σας. Συνιστούμε να ακολουθήσετε [αυτόν τον οδηγό](https://realpython.com/installing-python/) για να ξεκινήσετε.

Αφού εγκαταστήσετε την Python, θα πρέπει να μπορείτε να εκτελείτε εντολές Python στο τερματικό σας. Μπορείτε να ξεκινήσετε εκτελώντας την παρακάτω εντολή για να βεβαιωθείτε ότι είναι σωστά εγκατεστημένη πριν προχωρήσετε στα επόμενα βήματα: `python --version`. Αυτό θα εκτυπώσει την έκδοση Python που είναι διαθέσιμη στο σύστημά σας.

Όταν εκτελείτε μια εντολή Python στο τερματικό σας, όπως `python --version`, πρέπει να σκέφτεστε το πρόγραμμα που εκτελεί την εντολή σας ως την «κύρια» Python στο σύστημά σας. Συνιστούμε να κρατάτε αυτή την κύρια εγκατάσταση χωρίς κανένα πακέτο, και να τη χρησιμοποιείτε για να δημιουργείτε ξεχωριστά περιβάλλοντα για κάθε εφαρμογή που εργάζεστε — με αυτόν τον τρόπο, κάθε εφαρμογή μπορεί να έχει τις δικές της εξαρτήσεις και πακέτα, και δεν θα χρειαστεί να ανησυχείτε για πιθανά προβλήματα συμβατότητας με άλλες εφαρμογές.

Στην Python αυτό γίνεται με [*εικονικά περιβάλλοντα*](https://docs.python.org/3/tutorial/venv.html), που είναι αυτόνομα δέντρα καταλόγων που περιέχουν κάθε ένα μια εγκατάσταση Python με μια συγκεκριμένη έκδοση Python μαζί με όλα τα πακέτα που χρειάζεται η εφαρμογή. Η δημιουργία ενός τέτοιου εικονικού περιβάλλοντος μπορεί να γίνει με διάφορα εργαλεία, αλλά θα χρησιμοποιήσουμε το επίσημο πακέτο Python για αυτόν τον σκοπό, που ονομάζεται [`venv`](https://docs.python.org/3/library/venv.html#module-venv).

Πρώτα, δημιουργήστε τον κατάλογο στον οποίο θέλετε να βρίσκεται η εφαρμογή σας — για παράδειγμα, ίσως θέλετε να δημιουργήσετε έναν νέο κατάλογο με όνομα *transformers-course* στο ριζικό κατάλογο του αρχικού σας καταλόγου:

```
mkdir ~/transformers-course
cd ~/transformers-course
```

Από μέσα σε αυτόν τον κατάλογο, δημιουργήστε ένα εικονικό περιβάλλον χρησιμοποιώντας το module `venv` της Python:

```
python -m venv .env
```

Θα πρέπει τώρα να έχετε έναν κατάλογο με όνομα *.env* στον κατά τα άλλα κενό φάκελό σας:

```
ls -a
```

```out
.      ..    .env
```

Μπορείτε να μπαίνετε και να βγαίνετε από το εικονικό περιβάλλον με τα scripts `activate` και `deactivate`:

```
# Ενεργοποίηση εικονικού περιβάλλοντος
source .env/bin/activate

# Απενεργοποίηση εικονικού περιβάλλοντος
deactivate
```

Μπορείτε να βεβαιωθείτε ότι το περιβάλλον είναι ενεργοποιημένο εκτελώντας την εντολή `which python`: αν δείχνει στο εικονικό περιβάλλον, τότε το έχετε ενεργοποιήσει επιτυχώς!

```
which python
```

```out
/home/<user>/transformers-course/.env/bin/python
```

### Εγκατάσταση εξαρτήσεων[[installing-dependencies]]

Όπως και στην προηγούμενη ενότητα για τη χρήση Google Colab, θα χρειαστεί τώρα να εγκαταστήσετε τα πακέτα που απαιτούνται για να συνεχίσετε. Και πάλι, μπορείτε να εγκαταστήσετε την έκδοση ανάπτυξης του 🤗 Transformers χρησιμοποιώντας τον διαχειριστή πακέτων `pip`:

```
pip install "transformers[sentencepiece]"
```

Είστε τώρα πλήρως ρυθμισμένοι και έτοιμοι να ξεκινήσετε!
