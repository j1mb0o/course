# Ώρα Εξέτασης!

Ήρθε η ώρα να δοκιμάσετε τις γνώσεις σας! Έχουμε ετοιμάσει ένα σύντομο κουίζ για να δοκιμάσετε την κατανόησή σας στις έννοιες που καλύφθηκαν σε αυτό το κεφάλαιο.

Για να συμμετάσχετε στο κουίζ, πρέπει να ακολουθήσετε αυτά τα βήματα:

1. Συνδεθείτε στον λογαριασμό σας Hugging Face.
2. Απαντήστε στις ερωτήσεις του κουίζ.
3. Υποβάλετε τις απαντήσεις σας.
